Instructions on how to execute the code:

1) Copy the files to local machine
2) Open command Prompt
3) Go to the path of file location
4) To execute Web Server code, type the following command on command prompt,
   
python server_code.py 8904

where 8904 is the port number . The syntax for this command is,
server_code_name < port_number >

5) To execute the Client code, type the following command on command prompt,

python client_code.py 127.0.0.1 8904 index.html

where 8904 is the port number . The syntax for this command is,
client_code < server_IPaddress >< port_no ><requested_file_name >
